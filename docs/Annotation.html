Overview of Annotation Handling

Annotation refers to any text which is associated with a Probe or Gene name. Annotation for a particular entity can be gathered from multiple sources (i.e databases like EMBL and SwissProt) and stored locally to reduce data loading times.

maxdView displays annotation in the Annotation Viewer . The viewer is activated by selecting "Display annotation" from the popup menu over a Spot in the main display. The viewer displays annotation from one or more sources in a series of panels.

Annotation data is cached as it is first loaded, this makes subsequent accesses faster. The cache can be controlled via the Annotation Loader options panel. This panel is opened using the "Options" button in the viewer.
