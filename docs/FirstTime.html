Welcome to maxdView

When maxdView starts for the first time, it looks through it's subdirectories for plugin modules. Please do not do anything until this process has been completed.

This is the online help system. You can press "Help" in any window or dialog box to display the relevant page in this browser.

This documentation can be searched, use the "Find" button in the top-left corner of this window.

To start with theory, try the Overview and Concepts documents.

In you prefer to get stuck in, begin with the Getting Started tutorial.
