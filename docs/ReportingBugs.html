Please submit Bug Reports

maxdView is a work in progess and is bound to contain bugs and features that are not supposed to be features.

Any bug reports, suggestions, improvements, ideas for future developments, or other constructive criticisms will be greatfully received.

Send general comments to maxd_info@cs.man.ac.uk.

Send bug reports to maxd_bugs@cs.man.ac.uk and please be sure to include the following information:

  • The version of maxdView that you are using

  • The name and version of the operating system you are using

  • The version of Java that you are using
    (use java -version to find this out)

  • A description of precisely what went wrong.
    (If maxdView has written an error log, please include it).

  • A (small) sample data set that illustrates the problem
