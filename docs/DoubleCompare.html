Numerical Comparisions

In additional to normal numerical values, there is a special value used for missing or unknown values and there are also special representations for positive and negative infinity. It is important to know how these special values interact with each other and with normal values when numerical comparisons are performed.

  • The value 'NaN' (not-a-number) is used to represent a value which is unknown or missing.

  • The values 'Infinity' and '-Infinity' represent positive and negative infinity respectively.

The following tables show how comparisons between these three special values and normal numerical values will be evaluated.

In each table, the rows contains the value on the left-hand side of the equation and the columns contain the value on the right-hand side.

For any comparison, e.g. "A = B", find the row corresponding to A and the column corresponding to B to see how the comparision will be evaluated.

For example, to discover the result of the test "Infinity > NaN": use the greater than table and check the intersection of the second row (for 'Infinity') and the third column (for 'NaN') to discover that the result is 'false' ; this means that 'Infinity' is not greater than 'NaN'.
